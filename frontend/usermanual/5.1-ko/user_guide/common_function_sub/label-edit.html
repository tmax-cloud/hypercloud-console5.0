레이블 추가/삭제

각 리소스에 레이블을 추가 및 삭제할 수 있다.

  1. 각 메뉴의 메인 화면에서 레이블을 수정할 리소스의 (액션) 아이콘을 클릭한다.

  2. 드롭다운 메뉴가 열리면 [레이블 수정]을 선택한다.

  3. 레이블 수정 팝업이 열린다. 이때 추가할 레이블를 입력한 후 <Enter> 키를 누르면 입력한 레이블이 추가된다. 만약 추가된 레이블을 삭제하려면 레이블 내용 오른쪽의 [×] 버튼을 클릭한다.

  4. 레이블의 추가 및 삭제가 완료되면 [저장] 버튼을 클릭해서 현재 내용을 저장한다.

☞ 더 알아보기
다음과 같은 방법으로도 리소스에 레이블을 추가 및 삭제할 수 있다.

  1. 각 리소스의 상세 정보 화면에서 오른쪽 상단의 [액션] 버튼을 클릭한다.

  2. 드롭다운 메뉴가 열리면 [레이블 수정]을 선택한다.
